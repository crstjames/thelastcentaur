The Last Centaur - Design Reference

This document serves as a design reference for the pixel art RPG style we're implementing for The Last Centaur.

Inspiration: Stein World

Stein World login screen with pixel art forest background and centered login form

Key UI Elements

  • Pixel art forest background with mountains
  • Large game title at the top
  • Wooden/parchment style UI panels
  • Tab-based navigation between login and register
  • Form fields with labels
  • Server selection dropdown
  • Action buttons with hover effects

Implementation Notes

For the background image, we need a high-quality pixel art forest scene. The image should be:

  • High resolution (at least 1920x1080)
  • Pixel art style with visible pixels
  • Forest scene with trees, possibly mountains in the background
  • Bright, vibrant colors that contrast with the UI

The UI elements should have:

  • Wooden/parchment appearance
  • Clear borders and shadows
  • Consistent padding and spacing
  • Text that's easy to read against the backgrounds

CSS Implementation

/* Background image */
.bg-image {
  background-image: url('/images/forest-background.jpg');
  background-size: cover;
  background-position: center;
  image-rendering: pixelated;
}

/* UI Panel */
.ui-panel {
  background-color: rgba(180, 83, 9, 0.9);
  border: 2px solid #d97706;
  border-radius: 8px;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.3);
}

/* Form inputs */
.input-field {
  background-color: #fef3c7;
  border: 1px solid #d97706;
  color: #78350f;
  padding: 8px 12px;
  border-radius: 4px;
}

/* Buttons */
.action-button {

  color: white;
  border: 1px solid #fbbf24;

  border-radius: 4px;
